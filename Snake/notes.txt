## Environment

Make sure that the environment is activated before you try to dowload any packages.
Once environemnt is activated in the outter folder, download packages should be no problem

Create an environment for Linux:
    python3 -m venv .venv

Activate environment for Linux:
    source .venv/bin/activate

Install packages:
pip install <package>

Deactivate environment:
deactivate
